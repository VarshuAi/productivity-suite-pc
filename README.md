<!-- ============================== SETUP ============================== -->

<h2>
<img src="https://media.giphy.com/media/LnQjpWaON8nhr21vNW/giphy.gif" width="28">
<samp>&nbsp;SETUP</samp>
</h2>

### Prerequisites
- [Node.js](https://nodejs.org/) v18+
- npm v8+

### Install & Run
```bash
# Install dependencies
npm install

# Start the app
npm start

# Development mode
npm run dev
```

### Build for Production
```bash
npm run build
```
